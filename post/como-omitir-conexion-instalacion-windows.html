Cómo evitar el mensaje "No tengo Internet" en la instalación de Windows

Durante la instalación de Windows 11, es común que aparezca la pantalla que exige conectarse a Internet antes de continuar. Esto impide crear una cuenta local, mostrando únicamente la opción de usar una cuenta Microsoft.

Para evitar escribir el comando OOBE\BYPASSNRO y permitir una instalación sin conexión, existen varios métodos seguros y sencillos. A continuación, te explico cómo hacerlo.

Método 1 — Modificar el instalador USB (recomendado)

Este método permite que el instalador de Windows ya tenga habilitada la opción de crear una cuenta local sin conexión a Internet.

  1. Conecta el USB de instalación a otra computadora.
  2. Abre la unidad USB y entra en la carpeta sources.
  3. Verifica si existe el archivo ei.cfg. Si no está, créalo con el Bloc de notas.
  4. Escribe dentro del archivo el siguiente contenido:
[Channel]
_Default

[VL]
1
  1. Guarda el archivo con el nombre exacto ei.cfg dentro de la carpeta sources.
  2. Retira el USB y úsalo normalmente para instalar Windows.

Resultado: El instalador permitirá continuar sin conexión y crear una cuenta local directamente, sin necesidad de escribir comandos adicionales.

Método 2 — Usar el comando desde la instalación

Si no deseas modificar el USB, puedes activar la opción “No tengo Internet” usando el comando manualmente.

  1. En la pantalla donde el instalador pide conexión a Internet, presiona Shift + F10.
  2. En la ventana del símbolo del sistema, escribe:
OOBE\BYPASSNRO
  1. Presiona Enter. El sistema se reiniciará automáticamente.
  2. Al volver al asistente, verás la opción “No tengo Internet” y luego “Continuar con configuración limitada”.

Método 3 — Crear el USB con Rufus

La herramienta Rufus permite crear instaladores de Windows con opciones automáticas para omitir restricciones.

  1. Descarga y abre Rufus.
  2. Selecciona la imagen ISO de Windows 11.
  3. Antes de crear el USB, Rufus mostrará opciones especiales:
  • “Eliminar requisito de conexión a Internet (cuenta Microsoft)”
  • “Omitir TPM y Secure Boot”
  1. Marca la opción para eliminar el requisito de conexión a Internet y crea el USB normalmente.

De esta manera, el instalador ya estará configurado para funcionar sin conexión y no mostrará el mensaje de conexión obligatoria.

Conclusión: Si deseas instalar Windows sin conectarte a Internet ni usar una cuenta Microsoft, lo más práctico es agregar el archivo ei.cfg dentro de la carpeta sources o crear el USB con Rufus utilizando la opción de omitir la cuenta en línea. Con cualquiera de estos métodos, podrás continuar con la instalación sin interrupciones.
